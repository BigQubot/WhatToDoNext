部署 teedoc 生成的网站

    由于 teedoc 生成的网页都是静态网页，所以直接按照常规的部署静态页面的方式部署即可

    teedoc生成的页面会在out目录

    使用teedoc serve会起一个HTTP服务，但是请不要使用在生产环境，是不可靠的

    对于生产环境，这里有几个简单实用的方法：

    • 部署到 github pages
    • 使用 nginx 部署到自建服务器
    • 使用 CDN 加速网站
