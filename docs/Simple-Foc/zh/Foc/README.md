## ODrive

ODrive是一个基于STM32的高性能FOC伺服驱动器，其可以同时驱动两个无刷电机，早期被大量使用与3D打印机或自制CNC车床电气系统的设计中，知名极客James Bruton使用其作为大部分自己制作机器人伺服驱动的核心单元，之前Stanford开源的8自由度四足机器人Doggo同样也采用了Odrive作为核心的驱动器。
